唉，终于组队了，终于可以只BB了，我就BB，我就不上！

和Xiper以及chenxh组队了～

下面是总结：

day1

```
第一天吃饱喝足，然后就上路了，我一开始就看到了C题大水题，但是我不敢想象这道题居然这么水……和队友BB了一下，然后就跳了，看其他题了，不然一血就是我的了
然后队友发现了A题，理论复杂度估计了一下，是10e8的，感觉比较玄，然后就没做，赛后事实证明是能过的
然后看了看绝世大贪心的E题，贪心了一个小时，和队友BB了一下，然后出了几组数据就过了
然后看了看I题，I题比较迷，就交给chenxh去做了，然后他想了5个小时，然后到比赛结束也没有想出来，最后问了问别人的AC做法，发现石自己读题读错了……
降低智商之i题= =
F题，大水题，但是不知道为什么没人做，然后我们就去做了= =
DFS TLE bfs ac = =

最后比赛结果还是比较满意的，排名还是感人。
```

day2

```
做题做题，一开始就看到水题了，开场10分钟就拍完了，队友问我，你这样傻暴力不会TLE么，我说，这是签到题，肿么可能卡人呢？交上去TLE了……

然后改成预处理一下，交上去就A了= =

然后看到了格雷码的题，我的天，这什么鬼，然后队友说他学过格雷码，然后啪啪啪讲完，就知道这是一个简单DP了，拍完就A了

然后看到字符串的题，拍完马拉车算法，然后N^2的复杂度，队友说，你这样傻暴力不会TLE么，我有个NlogN的算法，只是比较难写一点，你改改？我说，出题人不好卡，数据绝逼水，然后我就交了，然后就A了= =

然后看D题，傻逼线段树，可惜读错题了，赛后才发现读错了= =

I题，数学题，交给霄神做，霄神推了3小时公式，只差最后一点了，吃晚饭的时候直接顿悟了= =

对了，有道模拟题，我最多写了1/10，就写了150行，我感觉我走远了，就没敲了……

今天告诉我们，千万别读错题

前期一度RANK1，感觉自己无敌了，然后被干了= =，后面就是在挂机
```

day3

```
一开场就看到了D题，是一道数学题，然后窝这个数学鶸就放弃了，直接让队友来做了，然后队友啪啪啪把公式推好了，然后给我说要高精度，然后让我写JAVA，然后窝太弱了，拍了半小时java都没过编译，然后队友怒上C++模板就A了。

然后看到了J题，发现这道题被人水了过去，然而我们都知道别人是水过去的，但是我们自己就是水不过去，然后bitset优化了一下，就A了

E题，读题读错了，然后推奇怪的公式，然后队友就把这道题过了，那时候我在玩手指= =

H题，迷之构造题，给队友BB了一下题意，于是队友很快就嘴炮AC了，然后窝上去一拍一交就A了

C题，是一开始队友就告诉我，这场比赛应该不会有人过的神题，我闲的无聊，就把这道题读了，感觉是道傻逼题呀， 然后趁着他俩讨论的时候，就把这道题拍了，然后就过了样例和自己出的数据，然后谜一样的就交了，就A了……迷之AC，感觉自己写的非常傻逼

比赛还是一样，一度RANK1，在4小时比赛的时候，感觉已经艹了RANK2一题，感觉可以挂机了，于是就没认真想了，然后被人绝杀了2题……
```

day4

```
今天居然和老队员一起做题，好慌啊

一开始A题居然是A+B，但是会爆longlong，然后我们三个就吓傻了，我去拍了一发java的高精度，然后交了一发，交完这道题，发现我们是最后一名，慌得不行……

然后看了B题，B题是一个大暴力，在霄神的努力下，10e10的复杂度优化成10e9，交上去T了，然后再优化成10e8，然后过了，= =，谐的不行

在霄神优化C题的时候，我和沈爷在看D题，看见感觉是一个公式题，然后果断打表找规律，然后找到规律拍个逆元处理一下就吼了

然后看到最后一题，最后一题数据范围才100，感觉暴力就好了，然后就N^3暴力直接撸，但是会爆longlong，于是也套了高精度，但是高精度除法太慢了，居然会TLE，于是离线打表了 = =

居然打表打了半小时，谐的要死……

然后看了J题，J题是沈爷拍的，这道题一开始沈爷觉得是神题，然后沈爷想了想，这道题应该可以做，然后沈爷又想了想，这道题分明就是一道傻逼题，然后沈爷啪啪啪就过了……

I题是我和霄神在想，想了一个半小时，发现之前想的都是错的……GG

今天成绩还是蛮好的，主要是沈爷太强了，我这种套版流+嘴炮流选手，就只能BB水题了……
```

day5

```
前两个小时在梦游，我在做一道傻逼题，队友双双读错题，不知道在做什么

然后我和霄神位置换了换，霄神来拍傻逼题，我和沈爷在想题，然后趁着霄神思考的时候我和沈爷过了三道水题，霄神最后甚至都不知道我们过了这三道题……

霄神也顺利把B题这个水题过了

然后沈爷又使出了数位DP的强悍实力，过了F题，现在5题了，排名还可以

然后还剩下一个小时，我们目前还剩一道谜一样的计算几何，和一个二分图可以做，然后大家都觉得自己数学不错，不想去拍那个建图烦得要死的二分图，于是就开始肛计算几何了，顺利套了刘汝佳的模板

就顺利A了，这应该是我正式过得第一道稍微有一点算法的计算几何题，还是蛮开心的~
```

```
队友太强了

一开始看到神队过了，水题，于是我和霄神看到了，也跟着过了

然后沈爷顺便也啪啪啪就把模拟题A题A了

然后看到D题，霄神看到是日期与星期的关系，马上就说了，这是蔡勒公式。

我靠，这是什么公式，好神啊，O(1)由日期推星期的公式，好屌啊

于是啪啪啪也很快过了，现在我们是Rank2，好厉害！

然后霄神用一个简单的推论，就把这道题给A了

期间沈爷在想最小瓶颈生成树，一直TLE，正解是LCT= =

然后剩下2个半小时就一直在挂机了，排名一直掉啊掉= =

```
一开局，队友就开了A题，我说概率DP，然后就去想概率DP了，在队友想题的时候，我顺手把签到题A了
